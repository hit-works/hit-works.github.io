개인정보 처리방침

Heart Rush · 시행일 2026.04.14

에이치아이티(이하 '회사')는 「Heart Rush」(이하 '서비스') 이용자의 개인정보를 중요하게 여기며, 개인정보 보호법 및 관련 법령을 준수합니다. 본 방침은 서비스에서 어떤 정보를 수집·이용·보관하는지 안내합니다.

1. 수집하는 개인정보 항목

서비스는 별도 회원가입 없이 이용 가능하며, 다음과 같은 정보가 수집·처리됩니다:

  • 기기 식별자 (익명 ID) — 계정 생성 및 데이터 연동. 실명이나 이메일은 수집하지 않습니다.
  • 닉네임 — 랭킹, 확성기, 프로필 표시 (사용자가 직접 설정)
  • 게임 데이터 — 점수, 레벨, 보유 재화/아이템, 출석 기록
  • 확성기 메시지 — 사용자가 작성한 공개 메시지 및 신고 내역
  • IP 기반 국가 정보 — 확성기 지역 필터링 용도 (IP 자체는 저장하지 않음)

다음 데이터는 기기 로컬 저장소(localStorage)에만 저장되며, 서버로 전송되지 않습니다:

  • 게임 설정 (BGM/효과음 음소거 여부)
  • 일일 퀘스트 진행도, 출석 선물 수령 상태
  • 선물 뽑기 이용 기록

2. 제3자 서비스

서비스는 다음 제3자 서비스를 이용하며, 각 서비스로 전송되는 데이터는 아래와 같습니다:

  • Cloudflare, Inc. — 서버 호스팅, 데이터베이스(D1), KV 저장소. 서버 요청 로그, 게임 데이터, 닉네임이 전송됩니다. Cloudflare 개인정보처리방침 적용.
  • Google AdMob (앱 내) — 보상형 광고. Google 및 파트너사가 광고 제공 및 측정을 위해 광고 ID(GAID/IDFA), IP 주소, 기기 모델, 앱 상호작용 데이터를 수집할 수 있습니다. Google 개인정보처리방침 적용.

3. 이용 목적

  • 게임 서비스 제공 (점수 저장, 랭킹, 상점, 아이템 관리)
  • 일일 보상 시스템 운영 (출석 선물, 퀘스트, 선물 뽑기, VIP)
  • 광고 표시 (부활 시 보상형 광고)
  • 서비스 개선 및 오류 분석
  • 부적절한 확성기 메시지 신고 처리

4. 데이터 보안

서버와의 모든 데이터 통신은 TLS/HTTPS로 암호화됩니다. 업계 표준 보안 관행을 적용하여 데이터를 보호합니다.

5. 보유 및 파기

  • 서버 데이터: 서비스 이용 기간 동안 보유. 계정 삭제 요청 시 30일 이내 파기.
  • 로컬 데이터: 브라우저 데이터 삭제 또는 앱 삭제 시 자동 제거.
  • 제3자 서비스 데이터: 각 서비스의 개인정보처리방침에 따름.
  • 관련 법령에 따라 보존이 필요한 경우 해당 기간 동안 보관.

6. 제3자 제공

이용자의 개인정보를 판매하거나 제3자에게 제공하지 않습니다.

7. 이용자의 권리 및 계정 삭제

  • 계정 삭제: 앱 내 설정 > 계정 삭제 또는 이메일로 요청. 30일 이내 삭제.
  • 로컬 데이터 삭제: 브라우저 데이터 삭제 또는 앱 삭제.
  • 광고 개인화 거부: 기기 설정에서 광고 ID 재설정.
  • 개인정보 열람/정정: 이메일로 요청. 30일 이내 응답.

8. 아동 보호

본 서비스는 전체 이용가(IARC 3+)입니다. 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

9. 처리방침 변경

본 방침은 필요에 따라 변경될 수 있습니다. 변경 시 시행일 7일 전 공지합니다.
