개인정보 처리방침

Knock Knock · 시행일 2026.04.14

에이치아이티(이하 '회사')는 「Knock Knock」(이하 '서비스') 이용자의 개인정보를 중요하게 여기며, 개인정보 보호법 및 관련 법령을 준수합니다. 본 방침은 서비스에서 어떤 정보를 수집·이용·보관하는지 안내합니다.

1. 수집하는 개인정보 항목

서비스는 회원가입을 통해 이용하며, 다음과 같은 정보가 수집·처리됩니다:

  • 이메일 주소 — 계정 인증 및 로그인
  • 닉네임 및 프로필 사진 — 서비스 내 표시 (사용자가 직접 설정)
  • 메시지 내용 — 종단간 암호화(E2E) 전송. 서버에는 암호화된 형태로만 저장되며, 회사는 메시지 내용을 열람할 수 없습니다.
  • 기기 정보 (푸시 토큰) — 푸시 알림 발송 용도
  • 건강 데이터 (선택) — 걸음 수, 수면 시간 등. HealthKit/Health Connect에서 명시적 허용한 데이터만 수집. 서비스 제공 외 목적으로 절대 사용되지 않습니다.
  • 친구 관계 — 사용자가 직접 추가한 친구 목록

2. 제3자 서비스

서비스는 다음 제3자 서비스를 이용하며, 각 서비스로 전송되는 데이터는 아래와 같습니다:

  • Cloudflare, Inc. — 서버 호스팅, 데이터베이스(D1). 서버 요청 로그, 암호화된 메시지, 계정 정보가 전송됩니다. Cloudflare 개인정보처리방침 적용.
  • Apple Push Notification Service (APNs) / Google Firebase Cloud Messaging (FCM) — 푸시 알림 전송. 기기 푸시 토큰이 전송됩니다.
  • Apple HealthKit / Android Health Connect — 건강 데이터 조회. 사용자가 명시적 허용한 건강 항목만 기기에서 읽어 서버로 전송합니다. 광고·마케팅 목적 제3자 제공 없음.

3. 이용 목적

  • 메시지 전달 서비스 제공 (감사 메시지 작성, 전송, 수신)
  • 계정 관리 및 인증 (로그인, 프로필 관리)
  • 푸시 알림 발송 (새 메시지, 리마인더)
  • 건강 데이터 공유 (친구 간 건강 상태 확인)
  • 서비스 개선 및 오류 분석

4. 데이터 보안

서버와의 모든 데이터 통신은 TLS/HTTPS로 암호화됩니다. 메시지는 종단간 암호화(E2E)로 보호되어, 서버에서도 원문을 확인할 수 없습니다. 암호화 키는 사용자 기기에만 저장됩니다.

5. 보유 및 파기

  • 서버 데이터 (계정, 프로필, 친구 관계): 서비스 이용 기간 동안 보유. 계정 삭제 요청 시 30일 이내 파기.
  • 메시지: 수신 확인 후 서버에서 삭제. 암호화된 백업은 30일 보관 후 파기.
  • 건강 데이터: 최근 7일분만 서버에 보관, 이후 자동 삭제.
  • 로컬 데이터: 앱 삭제 시 자동 제거.
  • 관련 법령에 따라 보존이 필요한 경우 해당 기간 동안 보관.

6. 제3자 제공

이용자의 개인정보를 판매하거나 제3자에게 제공하지 않습니다. 건강 데이터는 광고, 마케팅, 데이터 브로커에게 절대 제공되지 않습니다.

7. 이용자의 권리 및 계정 삭제

  • 계정 삭제: 앱 내 설정 > 계정 삭제 또는 이메일로 요청. 30일 이내 삭제.
  • 건강 데이터 공유 중단: 앱 또는 기기 설정에서 권한 철회 시 즉시 중단.
  • 로컬 데이터 삭제: 앱 삭제 시 자동 제거.
  • 개인정보 열람/정정: 이메일로 요청. 30일 이내 응답.

8. 아동 보호

본 서비스는 만 14세 이상 이용자를 대상으로 합니다. 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

9. 처리방침 변경

본 방침은 필요에 따라 변경될 수 있습니다. 변경 시 시행일 7일 전 공지합니다.
